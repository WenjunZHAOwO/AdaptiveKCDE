# AdaptiveKCDE

This repo hosts the MATLAB script for manuscript "Adaptive Kernel Conditional Density Estimation":

https://math.nyu.edu/~tabak/publications/KCDE_Paper.pdf

## Example usage

For 1D example, try this script in MATLAB:
```
Main_1D_ultimate
```

For 2D example, try:
```
Main_2D_ultimate
```
